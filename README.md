## Get Started
1. Install packages.
```
composer install
```
```
npm install
```
2. Create .env file
```
cp .env.example .env
```
3. Generate hash key
```
php artisan key:generate
```
4. Configure database connection information in .env
5. Clear cash
```
php artisan config:cache
```
6. Run dev server
```
php artisan serve
```
```
npm run dev
```
